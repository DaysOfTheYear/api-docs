## Features

- **Role-based access control** - Four tiers (Free, Pro, Enterprise, Internal)
- **Automatic API key generation** - Keys are created automatically when you sign up
- **Comprehensive data** - Access to thousands of holidays and observances
- **Flexible queries** - Search by date, keyword, or identifier
- **Rich metadata** - Includes tags, countries, dates, and more

## Getting Your API Key

API keys are automatically generated when you create an account. No need to request accessâ€”it's available immediately!

1. Create an account on [Days Of The Year](https://www.daysoftheyear.com)
2. Navigate to your profile's Developer section
3. Copy your API key from the "Secret key" field

<Note>
All new users automatically receive Free tier access, which includes access to the `/today` and `/date` endpoints.
</Note>

## Base URL

All API requests should be made to:

```
https://www.daysoftheyear.com/api/v2
```
