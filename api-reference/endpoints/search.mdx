## Overview

Search for events by name or keyword. Returns matching events sorted by relevance.

<Warning>
This endpoint is **not available** to Free tier users. Requires Pro or Enterprise tier.
</Warning>

## Endpoint

```
GET /api/v2/search/{search}/
```

## Path Parameters

<ParamField path="search" type="string" required={true}>
  Search query. Can contain letters, numbers, spaces, hyphens, and URL-encoded characters.
  
  The search performs a sentence-based search, matching the entire query phrase.
</ParamField>

## Query Parameters

<ParamField body="limit" type="integer" required={false}>
  Limit the number of results returned. Maximum value is 10. If not specified or exceeds 10, defaults to 10.
</ParamField>

<ParamField body="timezone_offset" type="integer" required={false}>
  Timezone offset from UTC. Must be between -12 and 12. Defaults to 0 (UTC).
</ParamField>

## Access by Tier

| Tier | Access |
|------|--------|
| Free | ❌ Not available |
| Pro | ✅ Available |
| Enterprise | ✅ Available |

## Request Examples

### Basic search

```bash
curl -H 'X-Api-Key: YOUR_API_KEY' \
  'https://www.daysoftheyear.com/api/v2/search/coffee/'
```

### Search with limit

```bash
curl -H 'X-Api-Key: YOUR_API_KEY' \
  'https://www.daysoftheyear.com/api/v2/search/coffee/?limit=5'
```

### Search with spaces (URL encoded)

```bash
curl -H 'X-Api-Key: YOUR_API_KEY' \
  'https://www.daysoftheyear.com/api/v2/search/national%20coffee%20day/'
```

## Search Behavior

- **Sentence-based search**: The search matches the entire query phrase, not individual words
- **Relevance sorting**: Results are sorted by relevance using the Relevanssi search engine
- **Maximum results**: Limited to 10 results per query
- **Case-insensitive**: Search is case-insensitive
- All tiers only see published content

## Notes

- Search results are limited to a maximum of 10 items
- The search performs a sentence-based match, so "national coffee" will match "National Coffee Day" but may not match "Coffee National Day"
- Results are sorted by relevance, not by date or grade
- Empty search results return an empty `data` array with `response_count: 0`
